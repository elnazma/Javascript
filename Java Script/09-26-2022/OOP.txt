1. Object adalah data yang berisi 2 : field / properties / attributes dan method / function / behavior
2. Class adalah blueprint, prototype atau cetakan untuk membuat object
3. Class berisikan deklarasi semua properties dan function yang dimiliki object
4. Setiap object selalu dibuat dari Class
5. Sebuah Class bisa membuat object tanpa batas
6. Class tidak bisa langsung dipakai karena hanya cetakan, yang dipakai adalah objectnya
